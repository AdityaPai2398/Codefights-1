A string is said to be a correct sentence if it starts with the capital letter and ends with a full stop (.).

Given a string, check whether it represents a correct sentence.

Example

For 'This is a correct sentence.', the output should be true.

For 'this is an incorrect sentence.' or 'This is another incorrect sentence', the output should be false.

[input] string inputString

A non-empty string.

[output] boolean
