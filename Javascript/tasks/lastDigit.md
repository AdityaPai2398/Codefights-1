# Description
For given integers a and b, find the last digit of ab.

## Example
For `a = 2` and `b = 5`, the output should be

```javascript
lastDigit(a, b) = 2.
```

### Explanation:
 **2<sup>5</sup> = 32.**
- [input] integer a

  A positive integer.

- [input] integer b

  A non-negative integer.

- [output] integer

  The last digit of ab.
